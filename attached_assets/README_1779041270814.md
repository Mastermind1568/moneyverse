# Moneyverse

**The definitive financial education and masterclass platform.**  
Brutalist / editorial aesthetic. Uncompromising performance. Zero fluff.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | [Next.js 16](https://nextjs.org) — App Router |
| Language | TypeScript 5 |
| Styling | [Tailwind CSS v4](https://tailwindcss.com) — CSS-native, no config file |
| UI Components | [shadcn/ui](https://ui.shadcn.com) — New York style, CSS variables, zero radius |
| Animation | [Framer Motion v11](https://www.framer.com/motion/) |
| Icons | [Lucide React](https://lucide.dev) |
| Fonts | Playfair Display (serif · headings) + Inter (sans · body) via `next/font` |

## Design System

**Palette**

| Token | Hex | Usage |
|---|---|---|
| `--color-brand-white` | `#FFFFFF` | Backgrounds, inverse text |
| `--color-brand-black` | `#000000` | Primary text, borders, buttons |
| `--color-brand-orange` | `#F7931A` | Accent / CTA / Bitcoin brand colour |

**Typography**
- **Headings** — `Playfair Display`, serif, weight 700–900
- **Body** — `Inter`, sans-serif, weight 300–600

**Aesthetic rules**
- Border radius: `0px` everywhere — sharp corners, zero compromise
- Border color: `#000000` by default
- CTA hover: black → Bitcoin Orange

## Local Development

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Adding shadcn/ui Components

```bash
# Example: add the Button component
npx shadcn@latest add button

# Example: add a full set of form primitives
npx shadcn@latest add form input label
```

Components are output to `components/ui/` and respect the zero-radius design token automatically.

## Environment Variables

Create a `.env.local` file at the project root:

```env
# Auth (e.g. Clerk)
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

# Database
DATABASE_URL=

# Stripe (for payments)
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=
```
